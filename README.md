# ⏱️ Cronômetro

Um cronômetro simples e intuitivo desenvolvido com HTML, CSS e JavaScript. O projeto permite iniciar, parar e controlar a contagem do tempo de forma prática, sendo ideal para praticar manipulação do DOM, eventos e funções temporizadas em JavaScript.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## ✨ Funcionalidades

- ▶️ Iniciar a contagem
- ⏸️ Pausar o cronômetro
- ⏱️ Atualização do tempo em tempo real
- 📱 Interface simples, limpa e responsiva

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de JavaScript, como `setInterval`, `clearInterval`, manipulação do DOM e eventos de clique, além de reforçar a criação de interfaces utilizando HTML e CSS.
